# EduCloud Lite API Workspace

This folder contains API documentation, OpenAPI definitions, a runnable
Postman collection, sample request bodies, and the test plan.

## Postman

Import `postman/EduCloud.postman_collection.json`, start FastAPI on port `8001`,
and follow the Supabase/Postman instructions in `backend/README.md`. Course and
Lesson create requests store their returned IDs for the remaining requests.
